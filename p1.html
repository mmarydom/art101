Project 1

For this project, I was able to create something through code and actually understand what I am doing instead of copying and pasting things that our professor taught us. For this project, we were assigned to create our own canvas with drawing functions. I went towards a monster creation canvas rather than a program used for drawing. With using the tools provided from the sketch example, I decided to create functions that can help the artist add to a base. The canvas consists of a body base with two eyes. Key Q is a thick green stroke, key w is a red thick stroke, key e is a blue thick stroke, key r is yellow rectanlge for textured surface, and key t is an eye pattern. And if you ever want to restart everything, the key X clears the canvas.
